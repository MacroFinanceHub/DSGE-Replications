## Replications

[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)

This repository contains a selection of my replications. Some of the replications attempt to replicate only some of the the results in the respective papers, other contain extensions that are not featured.

  -   <sub> [Galí, J. (2020). Uncovered Interest Parity, Forward Guidance,
and the Exchange Rate](https://www.nber.org/papers/w26797).</sub> 

  - <sub> [Sims and Wu (2019). The Four Equation New Keynesian Model](nber.org/papers/w26067) with extensions to optimal monetary policy and forward guidance.</sub>
